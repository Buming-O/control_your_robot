### 通用的机械臂planner
由于好多厂家自带的planner很垃圾,所以我结合了RoboTwin2.0最新的planner,实现了一个通用接口.  
考虑到有些朋友还没有自己的机器人,所以顺便给出了RoboTwin中我们最喜欢的cobomagic双臂平台的操控示例啦.  
这个会慢慢做成一个仿真教程~
